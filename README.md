## 🧪 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/benjamin-nyankson/portfolio-react.git
cd portfolio-react
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
